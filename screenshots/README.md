# Screenshots

To add a screenshot of the widget:

1. Take a screenshot of the widget on your desktop
2. Save it as `widget.png` in this directory
3. Uncomment the line in README.md:
   ```markdown
   ![Widget Preview](screenshots/widget.png)
   ```
4. Commit and push:
   ```bash
   git add screenshots/widget.png README.md
   git commit -m "Add widget screenshot"
   git push
   ```
